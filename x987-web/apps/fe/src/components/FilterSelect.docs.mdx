# FilterSelect

Generic labeled Select for top-of-table filters. Controlled via `value`/`onChange`, styled with Ant Design + Tailwind, themed via `roles`.

## Usage

<Source
  code={`import { FilterSelect } from '@/components/FilterSelect'

function Example(){
  const [value, setValue] = useState('all')
  return (
    <FilterSelect
      label="Generation"
      value={value}
      onChange={(v) => setValue(String(v))}
      options={[
        { label: 'All', value: 'all' },
        { label: '987.1', value: '987.1' },
        { label: '987.2', value: '987.2' }
      ]}
    />
  )
}
`} language="tsx" />
