# ThresholdChip

Use a five-level chip to indicate desirability relative to a threshold. Levels: poor → weak → fair → good → excellent (best). Two light backgrounds for the most desirable (good/excellent), three darker for the rest (fair/weak/poor). Background opacity also steps from 100% at excellent down by ~15% per level.

## Usage

<Source
  code={`import { ThresholdChip } from '@/components/ThresholdChip'

<ThresholdChip color="teal" level="poor" text="poor" />
<ThresholdChip color="teal" level="weak" text="weak" />
<ThresholdChip color="teal" level="fair" text="fair" />
<ThresholdChip color="teal" level="good" text="good" />
<ThresholdChip color="teal" level="excellent" text="excellent" />
`} language="tsx" />
