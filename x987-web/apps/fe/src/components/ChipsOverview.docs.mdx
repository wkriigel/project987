# Chips Overview

Base: `Chip` â€” a low-level presentational token for compact labels.

Variants extend Chip to encode semantics and color tokens:

- `ThresholdChip`: visualizes a 5-level status mapped to `roles.threshold` scales.
- `PaintChip`: visualizes a paint sample + label for interior/exterior colors.
