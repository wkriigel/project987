# Color System

## Ramps

Each ramp is a series from 50 (lightest) to 900 (darkest). Use lighter tones for backgrounds and darker tones for text.

## Usage Guidelines

- Prefer role-based tokens (e.g., `bg.emphasis`, `text.muted`) mapped to palette values for consistency.
- Ensure sufficient contrast for text (WCAG AA). Avoid low-contrast text on tinted backgrounds.
- Reserve bright accents (teal-200/300) for data emphasis, not large background areas.
- Use neutral grays for surfaces; keep page backgrounds dark for a scan-friendly UI.
